# Protect

**Protect** — кроссплатформенный оффлайн-менеджер паролей, созданный на Electron.  
Все данные шифруются локально и никогда не покидают устройство пользователя.

## 🔐 Особенности

- AES-256-GCM шифрование
- Генерация ключа из мастер-пароля (PBKDF2)
- Полностью оффлайн работа
- Без облаков и сторонних серверов
- Генератор надёжных паролей
- Импорт и экспорт базы
- Кроссплатформенность (Windows / Linux / macOS)

## 🛡 Безопасность

Protect использует **Web Crypto API** для криптографии:

- **AES-256-GCM** для шифрования данных
- **PBKDF2** для derivation ключа из мастер-пароля
- Все данные шифруются **до записи на диск**

Никакая информация не отправляется в интернет.

## 🖥 Технологии

- Electron
- JavaScript
- Web Crypto API
- AES-256-GCM
- PBKDF2
